Airline Flight Data Analysis:

A comprehensive data visualization project using *Power BI* to analyze airline ticket prices, durations, and class distributions across various Indian airlines.

Key Highlights
* *Revenue Leader:* Vistara emerges as the airline with the highest sum of ticket prices.
* *Class Share:* Economy class dominates the market with over *78%* share.
* *Data Scale:* Analyzing flight data worth approximately *6 Billion* in total revenue.
* 
*  Tools Used:
* *Power BI* for Dashboarding & Data Visualization.
* *Microsoft Word* for detailed data reporting.

Repository Contents:
* Airline Dashboard.png: Visual representation of flight data.
* Airlines Data Report.docx: Detailed documentation and analysis.
