#  Airline Flight Data Analysis

## Project Overview
A comprehensive data visualization project using *Power BI* to analyze airline ticket prices, durations, and class distributions across various Indian airlines. This project aims to provide insights into pricing trends and flight availability.

## Key Insights & Highlights
- *Price Analysis:* Identified the cheapest and most expensive airlines based on class and timing.
- *Duration Trends:* Analyzed how flight duration impacts ticket pricing.
- *Class Distribution:* Visualized the availability of Economy vs. Business class across different routes.

##  Tools Used
- *Power BI:* For Data Visualization and Dashboarding.
- *Excel/CSV:* As the primary data source.
- *DAX:* For creating custom measures and calculated columns.
